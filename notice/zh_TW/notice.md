## Notice

### 包含眾多變更的重大更新（3.0.0）已發布！

部分功能的無障礙與易用性已獲改善。
```
在既有的側邊欄收合／展開切換按鈕下方新增了便利工具列。
現已提供查看公告的功能。
為快速檢視搜尋條件而設計的預覽功能，在某些情況下可能造成干擾。現已可關閉。此選項可於[設定]中調整，並可從工具列快速存取。
工具列已新增 [加入最愛] 按鈕。
```

由於 Chrome 擴充功能的帳號同步儲存空間預設僅 100KB，對最愛功能可能有所限制。
```
已提供可切換為使用 Google Drive 的選項。
位置：設定 → 效能。
```
